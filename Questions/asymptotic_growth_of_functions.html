  1. Enunciado: Prove que \((n + 1)² = O(n²)\)

    Demonstração:

    Temos que \(f(n) = (n + 1)²\) e \(g(n) = n²\).
    Pela definição de Big - O:

    \(\exists\) \(c\), \(n_0 > 0\) tal que \((n + 1)² \leq c\) \(*\) \(n²\), \( \forall n \geq n_0\)

    Desenvolvendo a desigualdade, temos que:

    \((n + 1)² \leq c\) \(*\) \(n²\)
    \(n² + 2n + 1 \leq c\) \(*\) \(n²\)

    Note que \(2n \leq 2n²\) para todo \(n \geq 0 \) e \(1 \leq n²\) para todo \(n \geq 1 \).
    Logo:

    \(n² + 2n + 1 \leq n² + 2n² + n² = 4n²\) para todo \(n \geq 1\)

    Assim, \(f(n) \leq c \) \(*\) \(g(n)\) para \(c = 4\) e \( n \geq n_0 = 1\).

    \(c\) \(*\) \(n² \leq n² -3n - 18 \)

    \(n²\) \(\leq 2n² -6n - 36 \)
    \( 0 \leq n² -6n - 36 \)

    \(\Delta = 6² - 4\) \(*\) \(1\) \(*\) \((-36)\)
