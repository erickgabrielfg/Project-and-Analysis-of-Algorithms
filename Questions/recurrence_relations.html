Questão 1

Enunciado: Utilize o método da substituição para calcular a complexidade de tempo de pior caso do algoritmo abaixo. No algoritmo abaixo, o parâmetro N da função é um número inteiro. A operação de divisão usada é a divisão inteira, ou seja, é a divisão que sempre resulta em um inteiro. 

Algoritmo(N):
   Se N == 0 entao
       Retorne 0
   Senao
       soma = 0
       Para j = 1, ..., N faça
           soma = soma + j
       Retorne soma + Algoritmo(N/2)

Demonstração:

Com base no algoritmo acima, temos os seguintes tempos de execução em cada linha:

1. \(\Theta(1)\)
2. \(\Theta(1)\)
3. \(\Theta(1)\)
4. \(\Theta(1)\)
5. \(\Theta(n)\)
6. \(\Theta(n)\)
7. \(T(n/2)\)

Somando os tempos de execução, temos:

\(T(n) = 4\Theta(1) + 2\Theta(n) + T(n/2)\) = \(T(n/2) + n\)

Obtemos a seguinte recorrência:

\( T(n) \begin{cases} \Theta(1), & \mbox{se } n = 0 \\ T(n) = T(n/2) + n, & \mbox{se } n > 0 \end{cases} \)

Chutemos que \(T(n) = \Theta(n)\).
Assim, queremos provar que a solução para a recorrência \(T(n) = T(n/2) + n\) é \(\Theta(n)\).

Prova do limitante superior:

Seja \(T(n) = T(n/2) + n\).
Queremos provar que \(T(n) \leq c \cdot n\), \(\forall\) \(n \geq n0\), onde c e n0 são constantes positivas.

Vou provar por indução em n.

Hipótese Indutiva: Suponha que \(\forall\) k com \(n0 \leq k \leq n\), \(T(k) \leq c \cdot k\), onde c e n0 são constantes positivas.

Passo Indutivo:

\(T(n) = T(n/2) + n \leq cn/2 + n = n(c/2 + 1)\)

\(n(c/2 + 1) \leq cn\)
\(c/2 + 1 \leq c\)
\(c + 2 \leq 2c\)
\(c \geq 2\)

Solução: c = 4 e \(n0 \geq 0\)

Caso Base:

Vamos escolher c = 4 e n0 = 0.
Para n = 1, temos:

\(T(1) = T(1/2) + 1 = 4 \cdot 1/2 + 1 = 3 \leq 4 \cdot 1 = 4\)

Solução: 1

Prova do limitante inferior:

Seja \(T(n) = T(n/2) + n\).
Queremos provar que \(T(n) \geq c \cdot n\), \(\forall\) \(n \geq n0\), onde c e n0 são constantes positivas.

Vou provar por indução em n.

Hipótese Indutiva: Suponha que \(\forall\) k com \(n0 \leq k \leq n\), \(T(k) \geq c \cdot k\), onde c e n0 são constantes positivas.

Passo Indutivo:

\(T(n) = T(n/2) + n \geq cn/2 + n \)

\(cn/2 + n \geq cn\)
\(n(c/2 + 1 - c) \geq 0\)

\(c/2 + 1 - c > 0\)
\(c/2 < 1\)
\(c < 1/2\)

Solução: c = 1/4 e \(n0 \geq 0\)

Caso Base:

Vamos escolher c = 1/4 e n0 = 0.
Para n = 1, temos:

\(T(1) = T(1/2) + 1 = 1/4\cdot1/2 + 1 = 9/8 \geq 1/4 \cdot 1 = 2/8\)

Solução: 1

Questão 2

Enunciado: Utilizando o método de substituição, prove limitantes superiores e inferiores assintóticos para as recorrências a seguir: 

       a. T(n)=T(n/2)+15, onde T(1)=1. 
       b. F(n)=2F(n/2)+n, onde F(1)=1. 
       c. G(n)=2G(√n), onde G(1)=1.

Demonstração:

a.

Chutemos que \(T(n) = \Theta(lgn)\).
Assim, queremos provar que a solução para a recorrência \(T(n) = T(n/2) + 15\) é \(\Theta(lgn\)).

Prova do limitante superior:

Seja \(T(n) = T(n/2) + 15\).
Queremos provar que \(T(n) \leq c \cdot lgn\), \(\forall\) \(n \geq n0\), onde c e n0 são constantes positivas.

Vou provar por indução em n.

Hipótese Indutiva: Suponha que \(\forall\) k com \(n0 \leq k \leq n\), \(T(k) \leq c \cdot lgk\), onde c e n0 são constantes positivas.

Passo Indutivo:

\(T(n) = T(n/2) + 15 \leq clg(n/2) + 15 = c(lgn - lg2) + 15 = clgn -c + 15 \leq clgn\)

\(-c + 15 \leq 0\)
\(c \geq 15\)

Solução: c = 15 e \(n0 \geq 1\)

Caso Base:

Vamos escolher c = 15 e n0 = 2.
Para n = 2, temos:

\(T(2) = T(1) + 15 = 15\cdot lg1 + 15 = 15 \leq 15 \cdot lg 2 = 15\)

Solução: 2

Prova do limitante inferior:

Seja \(T(n) = T(n/2) + 15\).
Queremos provar que \(T(n) \geq c \cdot lgn\), \(\forall\) \(n \geq n0\), onde c e n0 são constantes positivas.

Vou provar por indução em n.

Hipótese Indutiva: Suponha que \(\forall\) k com \(n0 \leq k \leq n\), \(T(k) \geq c \cdot lgk\), onde c e n0 são constantes positivas.

Passo Indutivo:

\(T(n) = T(n/2) + 15 \geq clg(n/2) + 15 = clgn - c + 15 \geq clgn\)

\(-c + 15 \geq 0\)
\(c \leq 15\)

Solução: c = 1 e \(n0 \geq 0\)

Caso Base:

Vamos escolher c = 1 e n0 = 1.
Para n = 1, temos:

\(T(1) = T(1/2) + 15 = clg(1/2) + 15 = 14 > clg(1/2) = -1\)

Solução: 1

b.

Chutemos que \(F(n) = \Theta(nlgn)\).
Assim, queremos provar que a solução para a recorrência \(F(n) = 2F(n/2) + n\) é \(\Theta(nlgn)\).

Prova do limitante superior:

Seja \(F(n) = 2F(n/2) + n\).
Queremos provar que \(F(n) \leq c \cdot nlgn\), \(\forall\) \(n \geq n0\), onde c e n0 são constantes positivas.

Vou provar por indução em n.

Hipótese Indutiva: Suponha que \(\forall\) k com \(n0 \leq k \leq n\), \(F(k) \leq c \cdot klgk\), onde c e n0 são constantes positivas.

Passo Indutivo:

\(F(n) = 2F(n/2) + n \leq 2c(n/2)lg(n/2) + n = 2c(n/2)(lgn - 1) + n = cnlgn - cn + n \leq cnlgn\)

\(-cn + n \leq 0\)
\(n(1 - c) \leq 0\)

\(1 - c > 0\)
\(c < 1\)

Solução: c = 1/2 e \(n0 \geq 0\)

Caso Base:

Vamos escolher c = 1/2 e n0 = 1.
Para n = 1, temos:

\(F(1) = 2F(1/2) + 1 = lg(1/2) + 1 = 0 \leq 1/2 \cdot 1lg1 = 0\)

Solução: 1

Prova do limitante inferior:

Seja \(F(n) = 2F(n/2) + n\).
Queremos provar que \(F(n) \geq c \cdot nlgn\), \(\forall\) \(n \geq n0\), onde c e n0 são constantes positivas.

Vou provar por indução em n.

Hipótese Indutiva: Suponha que \(\forall\) k com \(n0 \leq k \leq n\), \(F(k) \geq c \cdot klgk\), onde c e n0 são constantes positivas.

Passo Indutivo:

\(F(n) = 2F(n/2) + n \geq 2c(n/2)lg(n/2) + n \geq c(n)(lgn - 1) + n = cnlgn -cn + n \geq cnlgn\)

\(-cn + n \geq 0\)
\(n(1 - c) \geq 0\)

\(1 - c < 0\)
\(c > 1\)

Solução: c = 2 e \(n0 \geq 0\)

Caso Base:

Vamos escolher c = 2 e n0 = 1.
Para n = 1, temos:

\(F(1) = 2F(1/2) + 1 = 4(1/2)lg(1/2) + 1 = - 2 + 1 = -1 \geq 2(1/2)lg(1/2) = -1\)

Solucão: 1

Questão 3

Enunciado: Usando o método da substituição, prove que a solução para a relação de recorrência T(n)=2T(⌊n/2⌋+17)+n é O(nlgn).

Demonstração:

Seja \(T(n)=2T(⌊n/2⌋+17)+n\).
Queremos provar que \(T(n) \leq c \cdot n \cdot lg n\), \(\forall\) \(n \geq n0\), onde c e n0 são constantes positivas.

Vou provar por indução em n.

Hipótese Indutiva: Suponha que \(\forall\) k com \(n0 \leq k \leq n\), \(T(k) = c \cdot k \cdot lg k\), onde c e n0 são constantes positivas.

Passo Indutivo:

\(T(n)=2T(⌊n/2⌋+17)+n \leq 2c(⌊n/2⌋+17)lg(⌊n/2⌋+17)+n \leq 2c((n+34)/2)lg((n+34)/2)+n\)

\(\leq c(n+34)(lg(n+34) - 1)+n \leq c(n+34)lg(n+34) - c(n+34)+n \leq c(n+34)lg(n+34)\)

O passo indutivo falha, pois não conseguimos dizer que \( c(n+34)lg(n+34) \leq cnlgn \).

Hipótese 2: \(T(k) \leq c (k - 34)lg(k - 34)\), onde \(\forall n0 \leq k \leq n\).

Passo Indutivo:

\(T(n)=2T(⌊n/2⌋+17)+n \leq 2c(⌊n/2⌋-34+17)lg(⌊n/2⌋-34+17)+n \leq 2c((n-34)/2)lg((n-34)/2)+n\)

\(\leq c(n-34)(lg(n-34) - 1)+n \leq c(n-34)lg(n-34) - c(n-34)+n \leq c(n-34)lg(n-34)\)

\(- c(n-34)+n \leq 0\)
\(34c - cn + n \leq 0\)
\(- cn + n \leq -34c\)
\(cn - n \geq 34c\)
\(n(c + 1) \geq 34c\)
\(n \geq 34c/(c+1)\)

Solução: c = 2 e \(n0 \geq 68\)

Questão 4

Enunciado: É possível provar que a solução para a recorrência T(n)=4T(n/3)+n é T(n)=Θ(\(n^{\log_34}\)). Mostre que uma prova de substituição que considere T(n)≤c\(n^{\log_34}\) falha. Depois, mostre como subtrair um termo de ordem mais baixa para fazer com que uma prova de substituição funcione. Basta provar apenas o limitante superior, não precisa provar o inferior.

Demonstração:

Irei desmonstrar que a solução para a recorrência \(T(n) = 4T(n/3) + n\) é \(\Theta(n^{log_34})\).

Seja \(T(n) = 4T(n/3) + n\).
Queremos provar que \(T(n) \leq c \cdot n^{log_34}\), \(\forall\) \(n \geq n0\), onde c e n0 são constantes positivas.

Vou provar por indução em n.

Hipótese Indutiva: Suponha que \(\forall\) k com \(n0 \leq k \leq n\), \(T(k) = c \cdot k^{log_34}\), onde c e n0 são constantes positivas.

Passo Indutivo:

\(T(n) = 4T(n/3) + n \leq 4c(n/3)^{\log_34} + n = 4cn^{\log_34} \cdot 3^{-\log_34} + n = 4cn^{\log_34} \cdot 4^{-1} + n = cn^{\log_34} + n\)

O passo indutivo falha, pois \(c \cdot n^{log_34} + n\) precisaria ser menor que \(c \cdot n^{log_34}\) e \(n\) é uma constante positiva. Assim, temos uma contradição.

Hipótese 2: \(T(k) \leq c \cdot k^{log_34} -dk\), onde \(\forall n0 \leq k \leq n\) e \(d\) é uma constante positiva.

Passo Indutivo:

\(T(n) = 4T(n/3) + n \leq 4[c(n/3)^{\log_34} - dn/3] + n = 4[c(n^{\log_34}/4) - dn/3] + n = c(n^{\log_34}) - 4dn/3 + n \leq c(n^{\log_34})\)

\(-4dn/3 + n \leq 0\)
\(n(1 - 4d/3) \leq 0\)

\(1 - 4d/3 \leq 0\)
\(d \geq 3/4\)

Solução: a = 3/4 e \(n0 \leq 0\)
